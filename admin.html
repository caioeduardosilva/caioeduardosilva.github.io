Guia de Hospedagem (GitHub Pages)

  1. No GitHub, crie um novo repositório chamado caioeduardosilva.github.io (ou dê outro nome se for enviar para subdiretório).
  2. Faça o upload/commit de todos os arquivos desta pasta para o repositório (`index.html`, `blog.html`, `style.css`, `posts.json`, pasta `posts`).
  3. No repositório, vá em Settings > Pages e garanta que está ativado para buscar da branch `main` ou `master` no root (`/`).
  4. Aguarde alguns minutos e seu site estará no ar! O GitHub trata tudo como site estático nativamente, e seu código já é 100% puro (sem build necessário).
